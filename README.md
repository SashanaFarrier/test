# Simple Calculator

A basic calculator application that performs simple arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Performs addition
- Performs subtraction
- Performs multiplication
- Performs division

## Usage

1. Clone the repository:
```bash
git clone https://github.com/yourusername/simple-calculator.git
```

2. Navigate to the project directory:
```bash
cd simple-calculator
```

3. Run the calculator:
- If using a Python script, execute:
```bash
python calculator.py
```
- Follow the on-screen prompts to input numbers and select operations.

## Example
```
Enter first number: 8
Enter second number: 4
Choose operation (+, -, *, /): *
Result: 32
```

## Notes
- Ensure valid inputs are entered when prompted.
- For division, inputting zero as the second number will display an error message.
